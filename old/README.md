## CanvasliteTS について

* [toile.js](https://github.com/TakashiNishimura/toile.js) の原型（2013-2014年に開発）
* TypeScript で記述した HTML5 Canvas （CanvasRenderingContext2D） 制御用のライブラリ
* より高機能な [toile.js](https://github.com/TakashiNishimura/toile.js) の利用を推奨
* 開発の再開はなし
